1.mo9的接入平台划分
   mo9支付系统共分四组服务器平台，具体划分如下：
   mo9.com            :国际生产环境，为中国大陆以外的其他国家和地区提供支付服务，包括中国台湾/中国香港。
   sandbox.mo9.com    :国际沙箱测试环境，为mo9.com提供商务对接测试平台

   mo9.com.cn         :中国生产环境，为中国大陆地区提供支付服务，不包含中国台湾、中国香港。
   sandbox.mo9.com.cn :中国沙箱测试环境，为mo9.com.cn提供商户对接测试平台.

2.mo9集成账号：
   mo9集成账号将由mo9的商务(技术)人员发给你，账号由email,passwd,key三部分组成。根据你的接入计划，mo9的商务人员会发送给你一到四组集成账号，分别对应mo9的四个不同服务器平台。如果你尚未获取任何集成账号，请联系mo9技术人员索取.
   mo9集成账号使用如下：
   a)使用email，passwd登陆mo9商户后台，查看你的交易信息及统计。(登陆地址为：https://www.mo9.com/merchant,https://www.mo9.com.cn/merchant,https://sandbox.mo9.com/merchant,https://sandbox.mo9.com.cn/merchant.根据你账号所在平台，选择一个对应的地址即可.)
   b)使用email，key作为你集成接口中的pay_to_email和MD5签名KEY，完成支付集成工作.
   c)由于mo9个平台用户系统互不想通，请在登陆和接入时选择正确的平台和账号组合，以免操作失败。
